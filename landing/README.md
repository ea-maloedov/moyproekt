# МойПроект — Landing

Лендинг платформы **МойПроект**.

Минималистичный статичный HTML/CSS сайт с адаптивной версткой и scroll-анимациями.

## Запуск

Открыть `index.html` в браузере или поднять любой статичный сервер:

```bash
npx serve .
```

## Цель

Конвертировать руководителей IT-агентств и веб-студий в регистрации на платформу.

**Позиционирование:** «Клиенты видят всё сами. Вы работаете.»
